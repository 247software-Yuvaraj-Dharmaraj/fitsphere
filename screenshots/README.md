# Screenshots

Add screenshots of the live app here so they render in the root README.

Take them from the live site (https://fitsphere-two.vercel.app) — log in with a
demo account, then capture each page and save with these exact filenames:

- `dashboard.png` — Member Dashboard (charts, stat cards, best-time hint)
- `analytics.png` — Admin Analytics (log in as `admin@fitsphere.app`)
- `attendance.png` — Attendance page (streak + calendar)
- `slots.png` — Slot booking page

Tip: a clean browser window at ~1280px wide makes the nicest captures.
